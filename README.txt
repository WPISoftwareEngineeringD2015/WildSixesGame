To run SixesWild simply run the file WildSixesRunner.java

This will open our SplashScreen, and after some time you will see our Main Screen.
Clicking play will take you to the SelectScreen. 

Initially, you will not have access to all of the levels, but by clearing the ones available you will
be able to unlock more to play.

Pressing X closes the program.

When running all tests, user input is required exactly twice after games are won. Simply click the OK button that appears to allow the testing to finish. It should be at 80.7% coverage.
